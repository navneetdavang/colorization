# Black and White Image Colorization using Deep Learning
A deep learning project utilizing TensorFlow 2 to implement an AutoEncoder model for black and white image colorization. Achieved accurate colorization results through extensive training and optimization. Integrated the model into a Flask web application for user-friendly interaction and real-time colorization of grayscale images

## Technology:
- Python
- Google Colab
- Tensorflow2
- Flask

## Requirement
Python3

## Run the Application:

- Clone the Repository
```
git clone https://github.com/navneetdavang/colorization.git
```

- Go to the `app` directory
```
cd app
```
- Install the packages
```
pip install -r requirements.txt
```

- Add the AI model `Colorizer_ResidualAutoEncoder_1200_500.h5` in the `static\model\` directory
- Run the app using
```
flask run
```
